# Overview

This document contains optional functions for corporate users.

This page describes how to use these optional functions:

| Optional function | Overview |
| --- | --- |
| [Error notification](https://developers.line.biz/en/docs/partner-docs/error-notification/) | If the bot server returns an error when sending webhook events from the LINE Platform, you can notify a channel administrator by email. |
| [Provider page](https://developers.line.biz/en/docs/partner-docs/provider-page/) | A list of all the services that the [Provider](https://developers.line.biz/en/glossary/#provider) offers on the LINE Platform. |
| [Mission stickers](https://developers.line.biz/en/docs/partner-docs/mission-stickers/) | Mission stickers are provided to user upon completion of certain objectives. |
| [LINE notification messages](https://developers.line.biz/en/docs/partner-docs/line-notification-messages/overview/) | You can send messages to users by specifying their phone number, even without knowing their user IDs. |
| [LINE Profile+](https://developers.line.biz/en/docs/partner-docs/line-profile-plus/) | The user data such as name, gender, birthdate, phone number and address registered by the user in LINE Profile+ can be obtained from the service (web app) provided by provider. |
| [LINE Beacon](https://developers.line.biz/en/docs/partner-docs/line-beacon/) | This section describes the conditions for receiving LINE Beacon. |
| [Module](https://developers.line.biz/en/docs/partner-docs/module/) | This section describes the extensions provided by the LINE Marketplace. By using the module, even LINE Official Account that have not opened a Messaging API channel can use the advanced features of the Messaging API as an extension. |
| [Mark as read API (old)](https://developers.line.biz/en/docs/partner-docs/mark-as-read/) | "Read" can be displayed in all messages sent from a specific user. |

<!-- note start -->

**Use of optional functions requires an application**

Only corporate users who have submitted the required applications can use the functions described in this document. To use these functions with your LINE Official Account, contact your sales representative or contact [our Sales partners](https://www.lycbiz.com/jp/partner/sales/).

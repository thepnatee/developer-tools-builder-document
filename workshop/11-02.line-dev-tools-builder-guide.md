# LINE Developer Tools Builder - คู่มือการใช้งาน

> https://devtoolsbuilder.cloudea.tech/

---

## สิ่งที่ต้องเตรียมก่อนเริ่มใช้งาน

1. **LINE Official Account** - ต้องมี LINE OA อย่างน้อย 1 บัญชี
2. **LINE Developers Console** - เปิดใช้งาน Messaging API Channel แล้ว
3. **Channel ID & Channel Secret** - ได้จาก LINE Developers Console

---

## 1. การเข้าสู่ระบบ (Login)

### ขั้นตอน

1. เปิดเว็บไซต์ [devtoolsbuilder.cloudea.tech](https://devtoolsbuilder.cloudea.tech/)
2. กดปุ่ม **Login with LINE**
3. ระบบจะพาไปหน้า LINE Login เพื่อยืนยันตัวตน
4. อนุญาตให้แอปเข้าถึงข้อมูล Profile
5. เข้าสู่หน้า Dashboard เรียบร้อย

> ระบบใช้ LINE LIFF สำหรับ Authentication ร่วมกับ Firebase Auth

---

## 2. การเพิ่ม Channel (เชื่อมต่อ LINE OA)

หลังจาก Login แล้ว ต้องเพิ่ม Channel เพื่อเชื่อมต่อกับ LINE OA ของคุณ

### ขั้นตอน

1. ไปที่เมนู **Settings** (ตั้งค่า)
2. ในแท็บ **Channels** กดปุ่ม **"+ Add Channel"**
3. กรอกข้อมูล:
   - **Channel ID** - คัดลอกจาก LINE Developers Console
   - **Channel Secret** - คัดลอกจาก LINE Developers Console
   - **Channel Name** - ตั้งชื่อเพื่อจำง่าย (ไม่บังคับ)
4. กด **Save** เพื่อบันทึก
5. ระบบจะตรวจสอบ Credentials และเชื่อมต่อ Channel ให้อัตโนมัติ

### การสลับ Channel

- ที่ **Sidebar** ด้านซ้าย จะมี **Channel Selector** (Dropdown)
- เลือก Channel ที่ต้องการใช้งาน
- เครื่องมือทั้งหมดจะทำงานกับ Channel ที่เลือกอยู่

## 4. การส่งข้อความ (Send Message)

### ขั้นตอน

1. ไปที่เมนู **Messaging > Send Message**
2. เลือกรูปแบบการส่ง:
   - **Push** - ส่งถึงผู้ใช้ 1 คน (ใส่ User ID)
   - **Multicast** - ส่งถึงหลายคน (เลือกจากรายชื่อสมาชิก)
   - **Broadcast** - ส่งถึงผู้ติดตามทั้งหมด
   - **Narrowcast** - ส่งถึง Audience เฉพาะกลุ่ม
3. สร้างข้อความ:
   - พิมพ์ข้อความธรรมดา
   - เลือก Flex Message ที่บันทึกไว้
   - เพิ่ม Quick Reply
   - เพิ่ม Sticker
4. ดู **Preview** ข้อความบนหน้าจอจำลอง iPhone
5. กด **"Send Now"** เพื่อส่งทันที หรือ **"Schedule"** เพื่อตั้งเวลาส่ง (PRO)

### ดูประวัติการส่ง

- ไปที่ **Messaging > Message History**
- ดูสถานะการส่ง, จำนวนผู้รับ, วันเวลา
- ค้นหาและกรองตามเงื่อนไข

## 5. การสร้าง Rich Menu

### ขั้นตอน

1. ไปที่เมนู **Rich Menu > Rich Menu Builder**
2. อัปโหลดรูปพื้นหลัง (ขนาดแนะนำ: 2500x1686 หรือ 2500x843 px)
3. วาดพื้นที่คลิกได้ (Clickable Area) บนรูป (สูงสุด 20 พื้นที่)
4. ตั้งค่า Action สำหรับแต่ละพื้นที่:
   - **Message** - ส่งข้อความ
   - **URI** - เปิดลิงก์เว็บ
   - **Postback** - ส่ง Postback Event ไปยัง Webhook
   - **Datetimepicker** - เปิดตัวเลือกวันเวลา
   - **Rich Menu Switch** - สลับไปเมนูอื่น
5. ตั้งค่าเมนู:
   - **Menu Name** - ชื่อเมนู
   - **Chat Bar Text** - ข้อความแสดงตอนยุบเมนู
   - **Set as Default** - ตั้งเป็นเมนูเริ่มต้นสำหรับทุกคน
6. กด **Create** เพื่อสร้างและ Deploy ไปยัง LINE

### จัดการ Rich Menu ที่มีอยู่

- **Rich Menu > Existing Rich Menus** - ดู, แก้ไข, ลบ Rich Menu
- **Rich Menu > Rich Menu Alias** - จัดการ Alias สำหรับสลับเมนู
- **Rich Menu > User Assignment** - กำหนดเมนูให้ผู้ใช้เฉพาะคน (PRO)

## 6. การสร้าง Flex Message

### ขั้นตอน

1. ไปที่เมนู **Messaging > Flex Message**
2. เลือกวิธีสร้าง:
   - **เขียน JSON** เอง ในช่อง JSON Editor
   - **เลือก Template** จาก Flex Showcase (20+ แบบ)
3. ดู **Live Preview** ทางด้านขวา (จำลองหน้าจอ iPhone)
4. ตรวจสอบ JSON ว่าถูกต้อง (ระบบแจ้ง Error ถ้าผิดรูปแบบ)
5. ใส่ **Title** และ **Alt Text**
6. กด **Save** เพื่อบันทึก Template
7. นำไปใช้ซ้ำในหน้า **Send Message** ได้

---

## 7. การสร้าง Imagemap

### ขั้นตอน

1. ไปที่เมนู **Messaging > Imagemap Builder**
2. อัปโหลดรูปภาพที่ต้องการ
3. กำหนดพื้นที่คลิก (Clickable Area) บนรูป
4. ตั้งค่า Action สำหรับแต่ละพื้นที่ (URI หรือ Message)
5. กด **Save** เพื่อบันทึก
6. ดู Imagemap ที่สร้างไว้ได้ที่ **Messaging > Existing Imagemaps**

## 8. การจัดการสมาชิก (Members)

### ขั้นตอน

1. ไปที่เมนู **Webhook > Members**
2. ระบบจะดึงรายชื่อผู้ติดตาม LINE OA มาแสดง
3. สิ่งที่ทำได้:
   - **ค้นหา** สมาชิกตามชื่อหรือ ID
   - **ดูรายละเอียด** - รูป Profile, ชื่อ, สถานะ, วันที่ติดตาม
   - **Tagging** - ติดแท็กสมาชิกเพื่อแบ่งกลุ่ม
   - **ส่งข้อความ** ถึงสมาชิกโดยตรง
   - **Refresh Profile** - อัปเดตข้อมูลจาก LINE
4. **Bulk Operations:**
   - Import สมาชิกจากไฟล์ CSV
   - Export รายชื่อเป็น CSV
   - Tag สมาชิกหลายคนพร้อมกัน

## 9. Chat Mode (แชทสด)

> ต้องใช้ Package PRO ขึ้นไป

### ขั้นตอน

1. ไปที่เมนู **Webhook > Chat**
2. ดูรายการสนทนาทั้งหมดทางด้านซ้าย
3. คลิกเลือกสนทนาเพื่อดูข้อความ
4. พิมพ์ตอบกลับในช่องด้านล่าง
5. ใช้ **Quick Actions** เพื่อ:
   - ส่งสินค้า (Product Carousel)
   - ส่งลิงก์จอง (Booking Link)
   - แจกคูปอง
6. เพิ่ม **โน้ต** หรือ **Reminder** ในแต่ละบทสนทนา
7. ใช้ **Response Templates** สำหรับข้อความที่ใช้บ่อย

## 10. Webhook Auto-Responses (ตอบกลับอัตโนมัติ)

> ต้องใช้ Package PRO ขึ้นไป

### ขั้นตอน

1. ไปที่เมนู **Webhook > Webhook Responses**
2. กด **"+ Create Rule"** เพื่อสร้างกฎใหม่
3. ตั้งค่า:
   - **Keyword/Trigger** - คำที่ต้องการให้ระบบตอบ
   - **Response** - ข้อความตอบกลับ (Text, Flex, Sticker)
   - **Workflow Group** - จัดกลุ่มกฎ
4. เปิด/ปิดกฎได้ด้วย Toggle
5. ดูสถิติว่ากฎไหนถูก Trigger กี่ครั้ง

## 11. Bot Templates (แม่แบบบอท)

### ขั้นตอน

1. ไปที่เมนู **Utilities > Bot Templates**
2. เลือก Template ที่ต้องการ (มีให้เลือกมากกว่า 11 แบบ)
3. กด **Preview** เพื่อดูตัวอย่าง
4. กด **"Create Bot"** เพื่อสร้าง
5. ปรับแต่งผ่าน **Customization Wizard**:
   - แก้ไขข้อความต้อนรับ
   - ตั้งค่า Keyword
   - เลือก Response Template
6. กด **Deploy** เพื่อเปิดใช้งาน

## 12. Live Polling (โหวตสด)

> FREE: สร้างได้ 5 โพล, ผู้โหวตสูงสุด 40 คน/โพล | PRO: 20 โพล, 100 คน/โพล | BIZ: 50 โพล, ไม่จำกัดผู้โหวต

### ขั้นตอน

1. ไปที่เมนู **Utilities > Live Polling**
2. กด **"+ Create Poll"**
3. ตั้งคำถามและตัวเลือก (Multiple Choice)
4. กด **Start** เพื่อเริ่มโหวต
5. แชร์ลิงก์ให้ผู้โหวตผ่าน LIFF
6. เปิด **Presenter Mode** เพื่อแสดงผลบนจอใหญ่
7. ดูผลโหวตแบบ **Real-time**
8. **Export** ข้อมูลการโหวตเมื่อเสร็จ

## 13. Quiz Game (เกมตอบคำถาม)

> FREE: สร้างได้ 3 เกม, ผู้เล่นสูงสุด 30 คน/เกม | PRO: 10 เกม, 100 คน/เกม | BIZ: 30 เกม, 1,000 คน/เกม

### ขั้นตอน

1. ไปที่เมนู **Utilities > Quiz**
2. กด **"+ Create Quiz"**
3. เพิ่มคำถาม:
   - **Multiple Choice** - ตัวเลือกหลายข้อ
   - **True/False** - ถูก/ผิด
   - ตั้งเวลาตอบแต่ละข้อ (Timer)
4. กด **Start Session** เพื่อเริ่มเกม
5. แชร์ **PIN Code** ให้ผู้เล่นเข้าร่วมผ่าน LIFF
6. ใช้หน้า **Host** ควบคุมการเล่น (เปลี่ยนคำถาม, ดู Leaderboard)
7. ผู้เล่นตอบผ่านหน้า **Player** บนมือถือ
8. ดูคะแนนและ Leaderboard เมื่อจบเกม

## 15. Utility Tools

1. ไปที่ **Utilities > Image Uploader**
2. เลือกรูปภาพ
3. ตั้งวันหมดอายุ (ถ้าต้องการ)
4. กด Upload - ได้ URL สำหรับใช้งาน

### QR Generator
1. ไปที่ **Utilities > QR Generator**
2. ใส่ URL หรือข้อมูล
3. ระบบสร้าง QR Code ให้ทันที

### PromptPay QR
1. ไปที่ **Utilities > PromptPay QR**
2. ใส่เบอร์โทรศัพท์
3. ใส่จำนวนเงิน (ถ้าต้องการ)
4. ได้ QR พร้อมเพย์ทันที

### Short Links (ย่อลิงก์) - PRO
1. ไปที่ **Utilities > Short Links**
2. ใส่ URL ที่ต้องการย่อ
3. ตั้ง UTM Parameters (ถ้าต้องการ)
4. ได้ลิงก์สั้นพร้อม Click Analytics

### Image Resize
1. ไปที่ **Utilities > Image Resize**
2. อัปโหลดรูปภาพ
3. เลือกขนาดที่ต้องการ (ตามข้อกำหนดของ LINE)
4. กด Download รูปที่ปรับขนาดแล้ว

### LINE URL Scheme
1. ไปที่ **Utilities > LINE URL Scheme**
2. เลือกประเภท Deep Link ที่ต้องการ
3. ระบบสร้าง URL ให้อัตโนมัติ
4. คัดลอกไปใช้งาน

## 16. Webhook Proxy & Event Logging

> ต้องใช้ Package PRO ขึ้นไป

### ขั้นตอน

1. ไปที่เมนู **Webhook > Webhook Proxy**
2. ตั้งค่า **Forward URL** - Endpoint ที่ต้องการให้ส่ง Event ไป
3. ระบบจะ Proxy Webhook Events จาก LINE ไปยัง Endpoint ของคุณ
4. ดู **Event Log** ทั้งหมดที่ได้รับ
5. ดู **สถิติ Webhook** - จำนวน Event, ประเภท Event

## 17. Cronjob Scheduler (ตั้งเวลาทำงาน)

> ต้องใช้ Package PRO ขึ้นไป

### ขั้นตอน

1. ไปที่เมนู **Utilities > Cronjob**
2. กด **"+ Create Cronjob"**
3. ตั้งค่า:
   - **Cron Expression** - กำหนดรอบเวลา (เช่น `0 9 * * *` = ทุกวัน 9 โมง)
   - **Webhook URL** - Endpoint ที่จะถูกเรียก
   - **Payload** - ข้อมูลที่จะส่ง
4. กด **Test** เพื่อทดสอบ (Dry-run)
5. กด **Save** เพื่อเปิดใช้งาน
6. ดู **Execution Log** เพื่อตรวจสอบการทำงาน

## 18. การเชิญสมาชิกทีม

### ขั้นตอน

1. ไปที่ **Settings > Members**
2. กด **"Invite Member"**
3. ระบบสร้าง **Invite Link** ให้
4. ส่งลิงก์ให้สมาชิกทีม
5. สมาชิกเปิดลิงก์ → Login ด้วย LINE → กด **"Join Channel"**
6. สมาชิกจะเข้าถึง Channel ที่ได้รับเชิญ

### การจัดการสิทธิ์

| Role | สิทธิ์ |
|------|--------|
| Owner | จัดการทุกอย่าง, เชิญ/ลบสมาชิก, ลบ Channel |
| Admin | จัดการเครื่องมือทั้งหมด, ดูสมาชิก |
| Member | ใช้เครื่องมือตามที่ได้รับอนุญาต |
